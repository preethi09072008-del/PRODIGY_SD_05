# PRODIGY_SD_05 - Web Scraping

## Description
This project is developed as part of the Prodigy InfoTech Software Development Internship (Task 5).

The program uses Python to scrape product information from the "Books to Scrape" website. It extracts details such as the book title, price, and availability, then stores the collected data in a CSV file for further use.

## Features
- Scrapes book details from a website
- Extracts title, price, and availability
- Saves the extracted data into a CSV file
- Uses BeautifulSoup for HTML parsing
- Uses Pandas to create the CSV file

## Technologies Used
- Python
- Requests
- BeautifulSoup4
- Pandas

## Output
The program generates a file named `books.csv` containing the scraped data.
